# TDA384

Repo for TDA384 (Principle of concurrent programming) at Gothenburg and Chalmers university.

## Trainspotting lab

### TSIM installation

```
./install_tsim.sh
```

### How to run:

```
cd ./trainspotting
make
make run
```
